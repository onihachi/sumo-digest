# 大相撲 取組ダイジェスト

日本相撲協会公式YouTubeチャンネルの取組動画を、日別・公開順に1本ずつ見られる静的サイトです。動画そのものはYouTubeのプライバシー強化モードで埋め込み表示します。

## ローカルで見る

```bash
npm run update
npm run check
npm run build
npm run dev
```

ブラウザで `http://localhost:4173` を開きます。

## 自動更新

`.github/workflows/update-videos.yml` が毎日18:00と19:00（日本時間）に公式チャンネルを確認し、`data/bouts.json` に更新があればコミットします。GitHub Pagesなど、リポジトリの更新を自動公開する静的ホスティングと組み合わせて使えます。

YouTube Data APIキーは不要です。チャンネルの公開動画一覧とタイトルを利用し、`＜令和○年○月場所・○日目＞` を含む通常動画だけを分類します。「幕下上位五番」の動画はプレイリストに含めません。

## 操作

- 大きな「次の取組」ボタンで次へ移動
- 前の取組ボタン、左右矢印キー、スマートフォンの左右スワイプに対応
- 上部のプルダウンで過去の日付へ移動
- 当日の動画がまだない場合、初期表示では最後に動画がある日を再生
- 動画がない当日をプルダウンで選ぶと公開待ち画面を表示
